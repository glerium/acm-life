## 7
发现一个很容易卡过去的做法：任意构造一个满足ABCD个数的抽卡序列（不考虑保底），随机打乱后直接输出，大概率可以直接通过本题

```cpp
void solve() {
    int n, x, y, z;
    cin >> n >> x >> y >> z;
    string s = string(n - x - y - z, 'A') + string(x, 'B') + string(y, 'C') + string(z, 'D');
    random_shuffle(s.begin(), s.end());
    cout << s << endl;
}
```
